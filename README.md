# Sozopol Sailing Cup – Official Project v1.0

This is the official static website project for Sozopol Sailing Cup.

## Upload to GitHub
Upload **all files and folders from inside this folder** to the repository root:

- `index.html`
- `en.html`
- `css/`
- `js/`
- `images/`
- `logos/`
- `docs/`
- `content/`
- `.nojekyll`
- `CNAME`

Do not upload the ZIP itself.

## GitHub Pages
After upload:
1. Open repository Settings.
2. Go to Pages.
3. Source: Deploy from a branch.
4. Branch: `main` / root.
5. Save.

Temporary URL will be like:
`https://pmomcheva-code.github.io/sozopol-sailing-cup/`

## Custom domain
The project includes a `CNAME` file for:
`sozopolsailingcup.com`

After GitHub Pages is enabled, update DNS in Namecheap as instructed by GitHub.

## Editing content
Main editable text is mirrored in:
- `content/bg.json`
- `content/en.json`

For now the HTML also contains the live text. After launch we can connect the JSON dynamically if you want fully content-driven editing.
